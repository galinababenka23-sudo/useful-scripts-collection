# Полезные скрипты

Сборник полезных скриптов для автоматизации задач.

## Установка

git clone https://github.com/galinababenka23-sudo/my-scripts.git
cd my-scripts

## Скрипты

### backup.sh
Скрипт для создания бэкапа.

Скрипт создаёт папку с датой и копирует туда файлы.
### clean-temp.sh
Очищает временные файлы старше 7 дней.
### monitor-cpu.sh
Показывает загрузку процессора в реальном времени.
