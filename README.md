# Полезные скрипты

Сборник полезных скриптов для автоматизации задач.

## Установка

git clone https://github.com/galinababenka23-sudo/my-scripts.git
cd my-scripts

## Скрипты

### backup.sh
Скрипт для создания бэкапа.

Скрипт создаёт папку с датой и копирует туда файлы.
### clean-temp.sh
Очищает временные файлы старше 7 дней.
### monitor-cpu.sh
Показывает загрузку процессора в реальном времени.
## Скрипты на Bash
### disk-usage.sh
Показывает использование диска.
### check-internet.sh
Проверяет соединение с интернетом.
### show-date.sh
Показывает текущую дату и время.
## Скрипты на Python
### hello.py
Выводит приветствие.
### calc.py
Выполняет простое сложение.
### show-time.py
Показывает текущее время.
## Как запустить
bash имя_скрипта.sh
python имя_скрипта.py
## Требования
- Linux или macOS
- Bash 4.0+
- Python 3.6+
